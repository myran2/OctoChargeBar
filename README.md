# OctoChargeBar
## What is this?
An addon to display charges of an important spell as a bar. Everything works in combat, but bars must be *initialized* out of combat in order to place tickmarks correctly.
### Example - Riptide
![A chargebar of the Riptide spell. The bar changes as charges are used.](./Images/example.gif)
## Spells Currently Tracked
* Augmentation Evoker
  * Hover
  * Prescience
* Devastation Evoker
  * Hover
* Preservation Evoker
  * Hover
* Fire Mage
  * Fire Blast
* Elemental Shaman
  * Lava Burst
* Restoration Shaman
  * Riptide
  * Lava Burst
## Configuration
All configuration happens in edit mode. Each bar for your spec will be enabled by default. You can turn off whatever you don't want and tweak whatever you do.

![The configuration window for the example Riptide chargebar above.](./Images/config.png)
